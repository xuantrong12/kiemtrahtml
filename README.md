# 🌐 Web Layout Project

Một dự án template website cơ bản được xây dựng dựa trên bản phác thảo giao diện giao diện người dùng (UI Mockup). Dự án này cung cấp một cấu trúc khung cấu hình chuẩn chỉnh bằng HTML5 và CSS Grid/Flexbox.

---

## 🗺️ Sơ đồ Giao diện (Layout Mockup)

Giao diện được hiện thực hóa dựa trên sơ đồ phân mảnh sau:
* **Header**: Thanh tiêu đề chính nằm trên cùng.
* **Navigation**: Thanh menu điều hướng bao gồm các mục: *Home*, *Section*, *Lesson*, và *Table*.
* **Main**: Khu vực hiển thị nội dung chính của trang web.
* **Footer**: Phần chân trang chứa thông tin cấu hình hoặc bản quyền.

---

## 🛠️ Công nghệ sử dụng

* **HTML5**: Định nghĩa cấu trúc ngữ nghĩa (`<header>`, `<nav>`, `<main>`, `<footer>`).
* **CSS3**: Định dạng giao diện, sử dụng **Flexbox** hoặc **CSS Grid** để phân chia bố cục chính xác theo bản vẽ.
